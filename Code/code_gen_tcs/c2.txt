**Code Generation test case 2: Computing expressions using integer and boolean data type**

**Single driver module with single level nesting **

<<<driver program>>>

	declare x, y, z:integer;
	declare a, b, c:integer;
	declare u, v: boolean;
	u:= true;
	a:= 5;
	b:= 9;
	get_value(x);
	get_value(y);
	z:= x + y*b +(a-b)*y+ a*2 - b*x;

	print(u);
end
